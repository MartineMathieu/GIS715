Styling maps with QGIS (Part 1)

Airports, flights and states data have been used to produced live layer effects. After uploading the Neon project, we used layer blending mode to change color fill, stroke width and size (airports). The live layer effect is produced by applying a "Drop shadow" to airports and a "Outer glow" to flight paths.

This second image used land, shaded relief and bathymetry data. We produced a live layer effect by checking "Draw effect" box for each of the bathymetry distance data. We also set the blending mode of the land data to "Multiply".
