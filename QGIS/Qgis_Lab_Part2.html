Styling maps with QGIS (Part 2)

For producing Layer blending Jemez, we used 4 different datasets: a DEM showing Jemez area, rivers, townships and different categories of land onwnership. The DEM has been displayed in Hillshade mode. We changed the Land ownership layer to the blending mode “Multiply”, “Overlay” for PLSS Townships layer. We then created a layout display of the data, adding the labels while making sure that labels don’t overlap map surrounds. The final result can be seeing above.

The image above uses Denmark OSM roads, Country boundaries and water datasets. The exercise uses blending mode to contrasting and highlighting the most dense area.

The image above uses USA states dataset. We run the topological coloring tool in order to create a map Shapeburst.
